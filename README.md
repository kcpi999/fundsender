# fundsender
Send funds to some other user. Laravel 5.

Simple as is: 
  register,
  
  authenticate, 
  
  find some other User on the site 
  
  and send him your money. Just... whatever amount you want. Or you able to.
  
---------------
Developed with:

Laravel 5.2

Bootstrap 3.3.6

jQuery 2.2.3

jQuery Validate 1.15.0

jQuery UI 1.12.0

Mysql 14.14
